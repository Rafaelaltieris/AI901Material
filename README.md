# Microsoft Learn - AI 901

Este repositório reúne anotações, exercícios, laboratórios e projetos desenvolvidos durante os estudos da trilha **Microsoft Learn AI Fundamentals**.

O objetivo deste material é consolidar conhecimentos em Inteligência Artificial utilizando os serviços da Microsoft Azure e Azure AI Foundry.

## Conteúdos Estudados

Durante esta trilha serão abordados conceitos fundamentais de IA, incluindo:

- Introdução à Inteligência Artificial
- Machine Learning
- IA Generativa
- Prompt Engineering
- Azure OpenAI
- NLP (Natural Language Processing)
- Computer Vision
- Speech AI
- AI Services
- Responsible AI
- AI Security
- Information Extraction
- Azure AI Foundry
