# Microsoft Learn - AI Fundamentals Studies and Labs

Este repositório reúne anotações, exercícios, laboratórios e projetos desenvolvidos durante os estudos da trilha **Microsoft Learn AI Fundamentals**.

O objetivo deste material é consolidar conhecimentos em Inteligência Artificial utilizando os serviços da Microsoft Azure e Azure AI Foundry.

## Conteúdos Estudados

Durante esta trilha serão abordados conceitos fundamentais de IA, incluindo:

- Introdução à Inteligência Artificial
- Machine Learning
- IA Generativa
- Prompt Engineering
- Azure OpenAI
- NLP (Natural Language Processing)
- Computer Vision
- Speech AI
- AI Services
- Responsible AI
- AI Security
- Information Extraction
- Azure AI Foundry

## 🚀 Plataforma Utilizada

- Azure Portal
- Azure AI Foundry
- Azure OpenAI
- Microsoft Learn
- GitHub
- Python
- VS Code

## Objetivo

Este repositório serve como:
- Material de estudo
- Registro de evolução
- Base para projetos futuros
- Apoio didático em aulas
- Preparação para certificações Microsoft AI Fundamentals
